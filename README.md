# AI Civic Issue Mapper

A government-style portal where citizens can report civic problems like garbage dumping, broken roads, water leakage, and street light failures.

## Features
- User registration and login
- Report civic issues with image and location
- AI-based issue classification
- Admin dashboard to manage complaints
- Issue tracking system

## Tech Stack
- Backend: Python (Flask)
- Database: MySQL
- Frontend: HTML, CSS, Bootstrap

## How to Run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Run the app: python app.py
4. Open browser: http://127.0.0.1:5000
